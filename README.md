## Consigne générale

Dans le cadre de ce cours, vous devez réaliser **un projet final en groupe**. Vous avez le choix entre deux projets distincts, chacun évaluant vos compétences sur l'utilisation des API REST.

Choisissez le projet qui correspond le mieux aux compétences de votre groupe, puis lisez attentivement le sujet avant de commencer.
